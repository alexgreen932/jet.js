Jet Ecosystem

Jet.css

  • A utility-first CSS framework — Tailwind-like, but simpler
  • Each class is logical and memorable .
  • Example: br-5 → border-radius: 5px
  • No custom CSS classes needed
  • Combine utilities and move on

Jet.js (Alpha)

  • A reactive framework inspired by Vue — but lighter, simpler, and purpose-built for Jet.Engine.
  • Designed to stay out of your way

Note: Current version uses Vue. Jet.js will fully replace it after testing.

MagicJet.js

  • A pure vanilla JS library for UI effects: sliders, accordions, animations, interactions.
  • No dependencies
  • Just works
